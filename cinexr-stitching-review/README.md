# CineXR Stitching Review

PowerSlides deck for reviewing the CineXR stitching test exports.

## Edit

- `slides.yaml` contains the slide order, copy, and per-slide media settings.
- `talk.js` contains the custom renderers for the comparison, media frame, title, and interactive 360 slides.
- `public/cinexr/` contains the deck media assets.

## Run Locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

The built static deck is written to `public/` and deployed to GitHub Pages by the repository workflow.
